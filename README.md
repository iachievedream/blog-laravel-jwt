### 下載安裝
~~~
git clone https://github.com/laravel/laravel.git
cd laravel
composer install 
copy .env.example .env
**set up MySQL of root and password
php artisan key:generate
php artisan jwt:secret
php artisan migrate:refresh
php artisan serve
~~~
